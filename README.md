## Projects

### Apply Filters to SQL Queries

**Location:**

`SQL-Projects/Apply-Filters-to-SQL-Queries/`

**Description:**

In this project, I used SQL to investigate potential security incidents involving login attempts and employee data.

I applied SQL filtering techniques using:

- AND operator
- OR operator
- NOT operator
- LIKE pattern matching
- Date filtering
- Time filtering

The investigation included analyzing failed login attempts, identifying suspicious login activity, and retrieving employee information required for security updates.

**Files:**

- `Apply-Filters-to-SQL-Queries.md` - Project documentation and explanations
- `queries.sql` - SQL queries used in the investigation
